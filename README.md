An implementation of [Ray Tracing in One
Weekend](https://raytracing.github.io/books/RayTracingInOneWeekend.html) in Haskell. The
Haskell is probably slow and bad. This is just for fun!

# Prequisites

Install [ghcup](https://www.haskell.org/ghcup/install/) to get cabal, ghc, etc..

# Running

`cabal run`
